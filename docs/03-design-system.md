# Design System

## Design Style

Zenith uses:
- minimalism
- soft elevation
- calm spacing
- premium typography

## Colors

Primary:
- Deep Teal

Secondary:
- Sage Green

Neutral:
- Warm Gray

Accent:
- Soft Gold

Danger:
- Muted Red

## Typography

Primary Font:
- Manrope

Fallback:
- Inter

## Border Radius

Cards:
- 24px

Buttons:
- 16px

Inputs:
- 14px

## Shadows

Use soft layered shadows.

Avoid harsh dark shadows.

## Cards

Cards are the main layout primitive.

Every card must:
- breathe
- avoid clutter
- contain one primary purpose
