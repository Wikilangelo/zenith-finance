# Frontend Architecture

## Stack

- Next.js App Router
- TypeScript
- TailwindCSS
- shadcn/ui
- Framer Motion

## Principles

- reusable components
- feature-based architecture
- strict typing
- responsive-first

## Structure

/app
/components
/features
/hooks
/lib
/services
/types
/styles

## UI Philosophy

Business logic must remain separated from presentation components.
