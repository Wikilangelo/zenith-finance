# Core Philosophy

## Gentle Intelligence

Zenith should never feel aggressive.

The AI must:
- guide
- suggest
- assist

Never:
- shame
- pressure
- overwhelm

## Calm Interface

The interface should feel:
- soft
- breathable
- minimal
- premium
- emotionally safe

## AI Transparency

Users must always understand:
- why an insight exists
- where data comes from
- what AI is doing

## Privacy First

Financial data is deeply personal.

Zenith must:
- minimize data collection
- explain AI processing
- offer export/delete options
- avoid dark patterns

## Human-Centered Finance

Zenith is not about maximizing wealth at all costs.

It is about:
- reducing stress
- improving awareness
- supporting goals
- creating healthy financial habits
