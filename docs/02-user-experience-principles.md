# UX Principles

## Mobile First

The mobile experience is the primary experience.

Desktop extends the experience but does not replace it.

## Progressive Disclosure

Avoid overwhelming users with too much information.

Complexity should appear gradually.

## Thumb-Friendly Navigation

Critical actions must remain reachable on mobile devices.

## Calm Design

Avoid:
- flashing colors
- aggressive red alerts
- excessive notifications

## AI Should Feel Silent

AI should feel ambient and intelligent.

Not noisy.

## Minimal Cognitive Load

Every screen must have:
- clear hierarchy
- limited actions
- obvious next steps

## Smooth Transitions

Navigation and animations must feel:
- fluid
- soft
- premium

Never:
- abrupt
- robotic
